# i3wm CONFIG

Minimal edit from i3wm default configuration.

## Things you need to install

- feh (*wallpaper display*)
- kitty (*terminal emulator*)
- polybar (*status bar*)
- picom (*transparency in terminal*)
- rofi (*program launcher/finder*)
